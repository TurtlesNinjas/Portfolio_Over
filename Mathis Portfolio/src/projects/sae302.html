Développer des applications communicantes (2024) – SAÉ302

  • Développement d’un quiz interactif utilisant PHP, HTML, CSS et JavaScript pour offrir une expérience utilisateur dynamique.
  • Utilisation de XAMPP et PHPMyAdmin pour gérer la base de données du quiz, avec Ngrok pour rendre le serveur local accessible publiquement.
  • Mise en place d’un système d’authentification sécurisé et d’un classement des joueurs.
  • Pratique complète du développement web front-end et back-end, intégrant gestion de base de données et accessibilité web.
